### Key invariants now enforced

- All inbox writes go through `createProposal` → `acceptProposal`. Update proposals can now also merge `newFrontmatter` (so the archive proposer can flip `status: archived` on accept).
- `cairndex sweep` runs `consolidate + archive` together; safe to run on every session end because both dedupe via SHA256 content hash.
- Stop hook in `init` template now writes two commands: `cairndex doctor --silent --auto-session` AND `cairndex sweep --silent`.
